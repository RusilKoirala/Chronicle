# Getting Started with Chronicle

Welcome to Chronicle! This guide will help you get up and running quickly.

## What is Chronicle?

Chronicle is a personal tracking application that helps you:
- Track achievements and milestones
- Save important resources and notes
- Set and monitor goals
- Manage daily tasks
- Build consistent routines

## Quick Start

### Web App
1. Visit [chronicle-app.com](https://chronicle-app.com)
2. Start adding your first achievement, goal, or task
3. No account required - your data is stored locally

### Mobile Apps
- **iOS**: Download from the App Store
- **Android**: Download the APK from our website

## First Steps

### 1. Add Your First Achievement
- Click "Add Achievement" on the dashboard
- Choose a type: book, certificate, skill, or other
- Add a title and description
- Set the completion date

### 2. Save a Resource
- Click "Save Resource" 
- Add notes, links, or file references
- Organize with categories and tags

### 3. Set a Goal
- Click "Set Goal"
- Define your objective
- Set a target date (optional)
- Track progress with the progress slider

### 4. Create Tasks
- Click "Add Task"
- Set due dates and priorities
- Mark as complete when done

### 5. Build Routines
- Click "Create Routine"
- Choose days of the week
- Set a time reminder
- Toggle active/inactive as needed

## Data Management

### Local Storage
- Your data is stored locally in your browser
- No account required
- Data persists between sessions

### Export/Import
- Export all your data as JSON
- Import data from backups
- Transfer between devices

### Privacy
- No tracking or analytics
- Your data never leaves your device
- Open source and transparent

## Tips for Success

1. **Start Small**: Begin with just one or two items
2. **Be Consistent**: Regular updates work better than bulk entries
3. **Use Categories**: Organize with tags and categories
4. **Review Regularly**: Check your dashboard weekly
5. **Export Backups**: Save your data periodically

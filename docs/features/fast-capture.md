# Fast Capture System

The Fast Capture System provides quick input methods for capturing tasks, goals, and other items.

## Implementation

See `src/components/fast-capture-system.tsx` for the main implementation.

## Features

- Quick capture modal
- Smart input with natural language processing
- Keyboard shortcuts
- Mobile-optimized input

## Usage

The fast capture system is accessible via:
- Floating action button (mobile)
- Keyboard shortcut (desktop)
- Quick capture button in navigation

## Components

- `QuickCaptureModal` - Main capture interface
- `SmartInput` - Intelligent input field with suggestions
- `FloatingActionButton` - Mobile quick access
